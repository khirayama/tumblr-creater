  • 上演（体育館）/ 18:00-

    『海のWindos、夜のDoors』

    （作・出演 / 島村和秀）

    30歳になった足立は、高校の同級生だったミカに出会う。舞台は瀬戸田。そして夏の夜。予感のすべてがここに！瀬戸田のスタンダードラヴが今明かされる！！

  • 上演（中庭）/ 19:30-

    『doolla』

    （2015年 / カラー / 35分）

    監督：市川玲

    広島県生口島で暮らす、青年ヨシトと妹のユカ、そして友人のケンイチ。何気ない日常を生きる彼らだったが、ある日ケンイチが行方不明となる。捜索に出たヨシトとユカが見るモノとは...?すべての撮影を瀬戸田で行った。純正瀬戸田映画。

  • 『どろん』

    （2014年 / カラー / 30分 / 嶺豪一 監督）

    高校生の時よく通っていた喫茶店のマスターが亡くなっていたことをネットで知った。僕の頭の中は急に高校時代に戻り、マスターの声が再生された。僕の携帯には今もマスターの携帯番号が残っている。どろん。

  • 『みんな蒸してやる』

    （2015年 / カラー / 41分 / 大河原恵 監督）

    かかしになりたいおとこは夢中 おんなは今更さよならふりむかせたい　私が結婚しても いつもの時間に きみがラジオきくなら
